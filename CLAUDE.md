## Documentation Roles

- `AGENTS.md` — AI agent-facing documentation (instructions, constraints,
  architecture, and operational rules for agents).
- `README.md` — Human-facing documentation (setup guides, usage instructions,
  and project overview for developers).
- Do not duplicate information between the two; each serves a distinct audience.
